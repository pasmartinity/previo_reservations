# Previo Reservations

Custom component for Home Assistant to fetch hotel reservations from Previo API.

## Installation

### Using HACS
1. Go to HACS > Integrations > Add Custom Repository.
2. Enter the repository URL and select Integration.
3. Search for "Previo Reservations" and install it.

### Manual Installation
1. Download or clone the repository.
2. Copy the `previo_reservations` folder to your Home Assistant `custom_components` directory.

## Configuration
Once installed, add the integration via the Home Assistant Integrations UI.
